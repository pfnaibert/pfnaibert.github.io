In this post, we will scrape the brazilian gdp data from the SIDRA-IBGE website. Here, I use my own functions (based on other codes) to download the sidra data, but be aware that theres is a CRAN package for that here. My source code for the functions can be found here. Also, there is another post here that makes a lot of the things we will be doing.

Ok, disclaimers are out of the way. First, let’s load up our functions and other libraries.


# source functions
source("../../R/funs-macro.R")

# load libraries
library(dplyr)
library(reshape2)
library(lubridate)
library(dygraphs)
library(ggplot2)
library(ggthemes)

The gdp.dl() function, downloads the data from the website and saves a .csv. The gdp.transform() loads the .csv file, using the paths used in gdp.dl(), transforms the data and saves a data.frame file and a xts file with the .rds format. I separated the gdp.dl() and the gdp.transform() functions to avoid connecting to the internet every time I wanted to edit the .csv file. Also I save a data.frame and a xts object because paged_table() and ggplot() do not accept xts objects, whereas dygraphs only accepts time series objects.


# download gdp
gdp.dl()

# transform gdp
gdp.transform()

To load the gdp data, I made to wrappers gdp.load.df() and gdp.load.xts(). This is to facilitate the loading and avoid typing the whole path.


# load gdp.df
gdp.df  <- gdp.load.df()

# load gdp.xts
gdp.xts <- gdp.load.xts()

# show gdp.df
paged_table(gdp.df)

Cool, the data is loaded, let’s make a table with the annual gdp data starting in 2014.


tmp <- filter(gdp.df, dates > "2014-01-01" & month(dates) == "12" )

Tables are ready, let’s graph the GDP percent change for better visualization in ggplot:


gdp.df %>%
filter(dates > "2014-01-01") %>%
select(dates, ret1, ret4, ret.ac4q.real) %>%
melt("dates") %>%
ggplot( aes( x = dates, y = value, fill = variable) ) +
geom_bar( stat="identity", position="dodge" ) +
scale_fill_discrete(name=" ",
                    breaks=c("ret1", "ret4", "ret.ac4q.real"),
                    labels=c("T/T-1", "T/T-4", "T/T-4 AC4Q") ) +
xlab("") + ylab("") +
ggtitle( "Brazilian GDP %Change" ) +
theme( plot.title = element_text(hjust = 0.5, face = "bold" ) ) +
theme_economist()

and interactive version with dygraphs:


tmp <- gdp.xts["2014/"]
dygraph(cbind(tmp$ret1, tmp$ret4, tmp$ret.ac4q.real),
        main="Brazilian GDP %Change") %>%
dySeries("ret1", label = "T/T-1") %>%
dySeries("ret4", label = "T/T-4") %>%
dySeries("ret.ac4q.real", label = "AC 4Q T/T-4") %>%

Now, let’s graph the level of the GDP using the Index of chained 1995 BRL, seasonally adjusted and not seasonally adjusted. In ggplot2:


den.SA  <- mean(gdp.df$real.SA[year(gdp.df$dates)=="2014"] )
den.NSA <- mean(gdp.df$real.NSA[year(gdp.df$dates)=="2014"] )

gdp.df %>%
select(dates, real.SA, real.NSA) %>%
mutate(real.SA = 100*real.SA/den.SA, real.NSA = 100*real.NSA/den.NSA ) %>%
melt(id = "dates") %>%
ggplot( aes(x = dates, y = value) ) +
  ggtitle( "GDP Index of 1995 Chained BRL (2014=100)" ) +
  geom_line(aes(color = variable), size = 1) +
  scale_color_manual(name = "", values = c("blue", "red"), labels=c("Seasonal Adj.", "Not Seasonal Adj." ) ) +
  theme( plot.title = element_text(hjust = 0.5, face = "bold" ) ) +
  xlab("") + ylab("") +
  theme_economist()

And with dygraphs:


tmp <- cbind( normalize.yr( gdp.xts$real.SA, "2014"), normalize.yr(gdp.xts$real.NSA, "2014" ) )
dygraph(tmp, main="GDP Index of 1995 Chained BRL (2014=100)" ) %>%
dyAxis("x", rangePad = 10, drawGrid = FALSE) %>%

dyEvent("2010-01-1", "Dilma", labelLoc = "bottom") %>%
